# Diler Uz - Admin Panel 🚀

## 🛠 Tech Stack
- **Frontend:** React.js (Vite)
- **Language:** TypeScript
- **UI Design:** Ant Design
- **Form Management:** Formik & Yup (Validation)

## ✨ Key Features
- **Product Management:** Full CRUD operations (Add/Edit/Delete).
- **Responsive Design:** Optimized for various devices.
- **Smart Tables:** Advanced filtering and sorting using Ant Design.
